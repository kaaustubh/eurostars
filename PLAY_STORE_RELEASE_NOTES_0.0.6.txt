Release 0.0.6:

New Features:
- Walk Mode Heatmap: Real-time visualization of plantar pressure with radial gradients.
- Calibration: Added dedicated calibration screens for left and right foot sensors.
- Session Management: New "Stop Session" dialog with options to Upload or Discard data.
- CSV Uploads: Session data is now uploaded in CSV format for easier analysis.
- Session History: Added "Gait Analysis" tab showing session history with retry logic for failed uploads.
- Data Sync: Session history now accurately reflects data stored in the cloud.

Improvements:
- Heatmap Accuracy: Precise taxel mapping and dynamic color scaling based on pressure (kPa).
- BLE Optimization: Connection priority set to LOW_POWER for battery efficiency.
- Reliability: Improved BLE connection stability and auto-reconnection logic.
- Security: Implemented anonymous authentication for secure patient data uploads.
