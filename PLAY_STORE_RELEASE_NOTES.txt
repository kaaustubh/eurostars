Release notes (update this file each release):

Release 0.0.8:

New Features:
- Lateral Center of Mass: New gait metric showing side-to-side body center of mass position.
- Balance Index: New balance metric based on Center of Pressure variability (lower variability = better balance).

Improvements:
- Expanded gait analysis from 5 to 7 metrics for more comprehensive analysis.
- Both new metrics available for clinicians and patients.

Release 0.0.7:

New Features:
- Session gait analysis detail screen for clinician and patient roles (tap a session to open).
- Session CSV v2 includes IMU (accelerometer + gyroscope) alongside pressure.

Improvements:
- Patient gait analysis detail is full-screen (tabs/header hidden) with back navigation.
- Chart indicator stays within the bar; 0 values render at the bar start.
- BLE scanning now detects all devices (removed UUID filtering for broader sensor discovery).
